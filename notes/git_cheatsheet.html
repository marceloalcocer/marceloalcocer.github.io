Git Cheatsheet¶

Branching¶

Merge changes in SOURCE into DEST:

git co <DEST>
git merge <SOURCE>

Rebase changes in SOURCE on to DEST:

git co <SOURCE>
git rebase <DEST>

Displaying¶

Show previous file revision [1]:

git show <REVISION>:<PATH>

Show all tracked files [2]:

git ls-tree -r master
[1]https://stackoverflow.com/questions/338436/is-there-a-quick-git-command-to-see-an-old-version-of-a-file
[2]https://stackoverflow.com/questions/15606955/how-can-i-make-git-show-a-list-of-the-files-that-are-being-tracked
