Strikethrough¶

No native strikethrough in ReST. Implant ourselves with a custom role and CSS [1]:

Define in the ReST document:

.. role:: strike
   :class: strike

Use in the ReST document:

:strike:`struck!`

In CSS:

.strike{
        text-decoration: line-through
}
